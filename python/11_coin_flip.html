# Control Flow


Every program we’ve seen so far has only had one possible path of execution — they all execute line by line, from top to bottom. And every time you run one of those programs, it gives you the same exact result. But it’s the twenty-first century, and we like options!

In the "Control Flow" section, we will explore how programs make decisions by evaluating conditions and introduce logic into our code!


# Instructions


Before we dive deep into if statement, let's do a demo!

Create a coin_flip.py program and type in the following:

          
          import random

          num = random.randint(0, 1)
             
          if num > 0.5: 
            print("Heads")
          else:
            print("Tails")

This program simulates a coin toss:

  • 50% of the time, it's Heads.
  • 50% of the time, it's Tails.

Run the program 5 times to see the if/else statement in action!

How many times did it go Heads?
