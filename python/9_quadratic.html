# User Input


When we are writing a program, sometimes we want the user to type in something

Python uses the input() method to get user input.

            
            username = input("Enter your user name:")

            print(username)

The output will say "Enter your user name:" and the user can type in something, hit enter and whatever the user typed gets stored into username.
