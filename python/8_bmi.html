# Exponention


Python can also perform exponentiation. In written math, you might see an exponent as a superscript number, but typing superscript numbers isn’t always easy on modern keyboards. Since this operation is so related to multiplication, we use the notation **.

            
            score = 2 ** 2      # score is 4
            score = 2 ** 3      # score is now 8
            score = 2 ** 4      # score is now 16
            score = 2 ** 5      # score is now 32
  
            print(score)        # Output: 32

# Instructions


The body mass index (BMI) is commonly used by health and nutrition professionals to estimate human body fat in populations.

It is computed by taking the individual's weight (mass) in kilograms and dividing it by the square of their height in meters.

\[bmi = {mass \over height²}\]
