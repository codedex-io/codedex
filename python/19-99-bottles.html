# Squares


The first program ever to run on a stored-program computer happened on May th, 1949 on the EDSAC to calculate and print a simple list of squares: 

          
            0   0
            1   1
            2   4
            3   9
            4   16
            5   25
            6   36
            7   49
            8   64
            9   81

On the left, there are numbers from 0 to 9. On the right are their squares. For example, in the last row, 9 * 9 = 81.

This can be created using string interpolation using the {} sign.

          
          for i in range(5):
            print("The square of " + i + " is " + i*i)
          
        
          
          The square of 0 is 0
          The square of 1 is 1
          The square of 2 is 4
