15. Sorting Hat (+15 XP)


# Congrats!


Here's a recap of everything we learned so far:

  • Control flow is the order in which the program's code executes.
  • if statement tests a condition for truth and executes the code if it is True.
  • elifclause can be added between if and else.
  • else executes the code if none of above is True.
  • Relational operators are used to compare two values: ==, !=, >, >=, <, <=.

Now let's put all your learnings together to create your own quiz!
