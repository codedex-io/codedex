# Comments


Comments are very important in your programs. They are used to tell you what something does in English, and they also are used to disable parts of your program if you need to remove them temporarily.

This is because the Python program ignores comments.

Here's how you use comments:

          
          # Printing out a message
          print("Hi")

In the above example, we created a comment using the # symbol. Because it's a comment, the output is still:

          
          Hi

Comments can also be placed more towards end of the line:

          
          print("Hi")   # I'm a comment, too!

Here too, the output would still just be "Hi".
