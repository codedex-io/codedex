15. Sorting Hat (+15 XP)


# Congrats!


Here's a review of what we learned:

  • Control flow is the order in which the program's code executes.
  • if statement test a condition for truth and execute the code if it is True.
  • elifclause can be added between if and else.
  • else execute the code if none of above is True.


# Instructions


The Sorting Hat is a talking hat at Hogwarts School of Witchcraft and Wizardry. It determines which of the four school Houses each new student belongs most to:

  • Gryffindor
  • Hufflepuff
  • Ravenclaw
  • Slytherin

Write a sortinghat.py program that asks the user some questions and places them into one of the four Houses based on their answers!

          
          Q1) Do you like Dawn or Dusk?
 
            1) Dawn
            2) Dusk
  • If answer is 1, one point to both Gryffindor and Ravenclaw.
  • Else if answer is 2, one point to both Hufflepuff and Slytherin.
  • Else, output the phrase "Invalid input".
          
          Q1) When I'm dead, I want people to remember me as:
 
            1) The Good
            2) The Great
            3) The Wise
            4) The Bold
  • If answer is 1, add one point to Hufflepuff.
  • Else if answer is 2, add one point to Slytherin.
  • Else if answer is 3, add one point to Ravenclaw.
  • Else if answer is 4, add one point to Gryffindor.
          
          Q3) Which kind of instrument most pleases your ear?
 
            1) The violin
            2) The trumpet
            3) The piano
            4) The drum
  • If the answer is 1, add one to Slytherin.
  • Else if the answer is 2, add one to Hufflepuff.
  • Else if the answer is 3, add one to Ravenclaw.
  • Else if the answer is 4, add one to Gryffindor.
  • Else, output Invalid input.
