# Random


In Python, we can use the .randint() function from the random module to generate a random number from a range.

But first, let’s import this module so we can use its functions.

          
          import random

Next, we’ll create a variable to store the randomly generated value. Declare a variable called random_number, and assign it to the function call: 

          
          random.randint(1, 9)

which will generate a random number between 1 (inclusive) and 9 (inclusive).

          
          import random

          num = random.randint(1, 9)

          print(num)

The output should be different each time.



# Instructions


The Magic 8-Ball is a popular toy developed in the 1950s for fortune-telling or advice seeking.

Create a magic8.py program that can answer any “Yes” or “No” question with a different fortune each time it executes.

We’ll be using the following 9 possible answers for our Magic 8-Ball:

  • Yes - definitely.
  • It is decidedly so.
  • Without a doubt.
  • Reply hazy, try again.
  • Ask again later.
  • Better not tell you now.
  • My sources say no.
  • Outlook not so good.
  • Very doubtful.

The output of should have the following format:

          
          Question: [Question]
          Magic 8-Ball: [Answer]

For example:

          
          Question: Is this real life?
          Magic 8-Ball: Better not tell you now.
