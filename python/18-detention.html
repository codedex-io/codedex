# range()


To loop through a set of code a specified number of times, we can use a for loop and the range() function.

The range() function returns a sequence of numbers, starting from 0 and increments by 1 (by default), and ends at a specified number. 

          
          for x in range(6):
            print(x)

This will output:

          
          0
          1
          2
          3
          4
          5

Notice how this stops at 5, and 6 is not printed.
