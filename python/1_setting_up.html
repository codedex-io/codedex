# Code Editor


Welcome to the very first task of The Legend of Python!

We know you are excited to get started, but before you write your first line of Python code, you need to grab yourself a code editor.

A code editor is a text editor where we can write and execute code. We recommend Visual Studio Code, created by Microsoft.




# Python


The programming language we are using is Python, which is created by Guido van Rossum in 1991.

Its design philosophy focuses on code readability, which makes it a wonderful coding language for beginners.

Python is also super versatile and is heavily used in:

  • Data Analysis
  • Data Visualization
  • Machine Learning
  • Natural Language Processing
  • Game Development
  • Web Scraping
  • And more!

All the code we write will be in Python files.

Audio files end in .mp3, GIF files end in .gif. Python files have the file extension .py.


# Instructions


  1. Download Visual Studio Code to your computer, whether it's a Mac or a PC.

  2. Watch this quick 5 min video: Get Started with Visual Studio Code.

  3. Create a new folder in your desktop and name it codedex.

  4. Inside the codedex folder, create an empty file and name it hello_world.py.

You are now ready for the journey ahead.

Happy coding!
