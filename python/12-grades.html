# If Statement


An if statement is used to test a condition for truth and execute the code if it is True: 

          
            if condition:
              # code inside

For example, if variable grade is greater than 60:

          
          if grade > 60:
            print("You passed!")

Note: The code "inside" the if statement must be indented (preferably 2 spaces).


# Else


An else clause can be optionally added to an if statement.

  1. If the condition evaluates to True, code in the if part is executed.
  2. If the condition evaluates to False, code in the else part is executed.
          
          if grade > 60:
            print("You passed.")
          else:
            print("You failed.")
